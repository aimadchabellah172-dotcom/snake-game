المتغيرات وأنواع البيانات Variables & Logic

💡 شرح دقيق للمفهوم:

المتغير في بايثون ليس مجرد "صندوق"، بل هو مرجع (Reference) لعنوان في ذاكرة الحاسوب (RAM). عندما تكتب x = 10، أنت تخبر بايثون أن تجعل الاسم x يشير إلى كائن رقمي قيمته 10.

ملاحظة احترافية: بايثون لغة "Dynamic Typing"، مما يعني أنك لا تحتاج لإخبارها بنوع البيانات (مثل int أو string)، هي تكتشف ذلك تلقائياً عند التشغيل!
variables.py
# بايثون تفهم النوع من القيمة مباشرة
score = 100          # int (عدد صحيح)
price = 19.99        # float (عدد عشري)
player_name = "علاء"  # str (نص)

# يمكنك تغيير النوع في أي وقت (غير مستحسن غالباً لكنه ممكن)
x = 5
x = "الآن أنا نص"

تطوير الألعاب باستخدام Pygame

🎮 ماهي Pygame؟

هي مكتبة برمجية مفتوحة المصدر مصممة لكتابة ألعاب الفيديو. تعتمد على مكتبة SDL وتسمح لك بالتحكم في الصور، الأصوات، وحركات لوحة المفاتيح والماوس بسهولة فائقة.

🛠️ التثبيت:

قبل البدء، يجب تثبيت المكتبة عبر Terminal باستخدام أداة pip:

pip install pygame

🔄 مفهوم "حلقة اللعبة" (Game Loop):

أي لعبة في العالم تتكون من حلقة تكرارية سريعة جداً (تصل لـ 60 مرة في الثانية) تقوم بـ 3 مهام:

  1. معالجة المدخلات: هل ضغط اللاعب على زر القفز؟
  2. تحديث الحالة: تحريك العدو، حساب الجاذبية.
  3. الرسم: مسح الشاشة ورسم العناصر في مواقعها الجديدة.
basic_game.py
import pygame

# 1. تهيئة بايثون جرافيكس
pygame.init()
screen = pygame.display.set_mode((800, 600))
clock = pygame.time.Clock()
running = True

# 2. حلقة اللعبة الأساسية
while running:
    # (أ) التحقق من الأحداث (Events)
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # (ب) التحديث والرسم
    screen.fill((30, 30, 30)) # لون الخلفية
    
    # رسم مربع لاعب (لون، أبعاد: x, y, width, height)
    pygame.draw.rect(screen, (0, 255, 0), (375, 275, 50, 50))

    pygame.display.flip() # تحديث الشاشة
    clock.tick(60)      # تحديد السرعة بـ 60 إطار في الثانية

pygame.quit()
تمرين: حاول تغيير الأرقام في screen.fill لتغيير لون الخلفية (نظام RGB).
